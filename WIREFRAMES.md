**Color Palette Used:**
- Background (parchment): #F0E6D3
- Primary accents: #C4006A (magenta), #8B1A52 (wine), #C8920A (gold), #00B4A6 (teal)
- Plum sections: #1E0A2E
- Card borders/separators: use section color + low opacity

**Typography:**
- Logo: Abril Fatface or Fraunces (as user selected)
- Headlines: Bold, larger sizes
- Body: Clean, readable serif or sans-serif
- CTA buttons: Bold, clear

**Interactive Elements:**
- Checkboxes/radio buttons: Large tap targets (44×44px minimum)
- Buttons: Full-width on mobile, contained width on desktop
- Modal: Smooth fade-in, close button always visible
- Progress bar: Visual feedback on quiz completion %

**Mobile Considerations:**
- Single column layout for all modals
- Large touch targets on form inputs
- Sticky CTA button at bottom
- Minimal scrolling per question
- Simplified card layouts
- Carousel for dog library on mobile (horizontal scroll)
